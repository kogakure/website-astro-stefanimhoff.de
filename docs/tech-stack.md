## Tech Stack

- **Framework**: Astro 6.x (static site generator with view transitions)
- **UI Libraries**: React, MDX for content
- **Styling**: Tailwind CSS v4.3 with `@tailwindcss/vite`; custom logical-property and OpenType utilities defined in `@utility` blocks in `src/styles/global.css` (no external plugins)
- **Search**: Pagefind (full-text search)
- **Package Manager**: pnpm
- **Build Tools**: Vite, TypeScript (strict mode)
- **Quality Tools**: ESLint, Prettier, Husky pre-commit hooks
