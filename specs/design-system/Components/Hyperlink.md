This is the styling for a regular hyperlink used in body text, e. g. in essays or other Markdown pages.

| State   | Style                                                   |
| ------- | ------------------------------------------------------- |
| Default | Beni (`#900B20`), no underline                          |
| Hover   | Beni (`#900B20`), underline draws in from left (200 ms) |
| Visited | Beni Muted (`#C75566`), no underline                    |
| Active  | Beni Dark (`#6A0818`)                                   |
