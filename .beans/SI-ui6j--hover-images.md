When the user hovers over the list of blog posts titles, a small thumbnail of the cover image of the post should be shown and stick to the mouses cursor as long as the user is hovering over the title. The thumbnail should disappear when the user moves the mouse away from the title. This feature will enhance the user experience by providing a visual preview of the blog post content, making it easier for users to decide which post to read.

In case that the blog post has no cover image, a default thumbnail should be shown instead.
