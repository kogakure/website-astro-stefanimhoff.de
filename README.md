# stefanimhoff.de

This is the source of my [personal website][si] build with [Astro], [Preact], and [Tailwind CSS].

## Installation

```sh
pnpm install
```

## Running the Website

```sh
pnpm dev
```

For more tasks have a look into the `package.json`.
